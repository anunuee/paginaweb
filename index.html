Matemáticas para Nuevo Ingreso

Proyecto de Titulación - Resumen de Contenido Programático

Introducción al Proyecto

Este programa está diseñado como una herramienta de nivelación académica para estudiantes de nuevo ingreso. El objetivo es reforzar las bases matemáticas mediante un sistema de evaluación continua dividido en tres periodos parciales, asegurando que el alumno domine las competencias necesarias para su trayectoria universitaria.

Subtemas: 5 Preguntas por Subtema: 50 Total Reactivos: 250 Evaluaciones: 3 Parciales

Estructura del Contenido

Parcial Subtema Descripción del Contenido Reactivos
1° Parcial 1. Aritmética Básica Operaciones con reales, fracciones, razones y proporciones. 50
2. Álgebra Elemental Leyes de exponentes, radicales y simplificación de expresiones. 50
2° Parcial 3. Ecuaciones y Funciones Sistemas de ecuaciones lineales, cuadráticas y modelado. 50
4. Geometría Euclidiana Perímetros, áreas, volúmenes y teoremas de semejanza. 50
3° Parcial 5. Trigonometría Funciones trigonométricas, identidades y resolución de triángulos. 50

Metodología de Evaluación

  • Banco de Preguntas: Cada subtema cuenta con 50 reactivos de opción múltiple con diferentes niveles de complejidad.
  • Criterio de Acreditación: Evaluación formativa tras finalizar cada bloque de subtemas.
  • Enfoque: Resolución de problemas aplicados a contextos reales.
